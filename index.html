Termux APT repository

This apt repository contains hugo packages for Termux under the ./dists/termux directory.

You can access the repo by adding a file at $PREFIX/etc/apt/sources.list.d containing the single line: 

deb [trusted=yes] https://holehan.github.io/termux-debs/ termux extras
As the repository is served via https, you may have to first install the apt-transport-https package.
