Termux APT repository for hugo

This apt repository contains hugo packages for Termux.

You can access the repo by adding a file ending in .list, e.g. holehan.list, at $PREFIX/etc/apt/sources.list.d containing the single line: 

deb [trusted=yes] https://termux.holehan.org/ termux extras
As the repository is served via https, you may have to first install the apt-transport-https package. The hugo package comes with bash-completion support and man pages. So you may want to install those packages as well in Termux: 
apt install bash-completion man
